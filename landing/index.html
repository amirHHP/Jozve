Jozve

Personal Knowledge Workspace

Capture knowledge before it disappears.

Jozve is a bilingual Chrome extension for capturing, organizing, and revisiting notes, ideas, and research.

View GitHub How to use

Why I built Jozve

I constantly collect ideas, references, articles, and fleeting thoughts while browsing. Most tools felt too heavy or too far from the actual workflow. Jozve was built as a lightweight personal knowledge layer that stays close to the browser.

Features

Quick Capture Popup
Full Dashboard Workspace
Source & Category Filters
Dark / Light Mode
English / Persian Support
Offline-first Storage

How to use

  1. Clone or download the GitHub repository
  2. Open Chrome Extensions
  3. Enable Developer Mode
  4. Click Load unpacked
  5. Select the extension folder
  6. Start capturing knowledge
